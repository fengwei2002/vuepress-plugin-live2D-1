# vuepress-plugin-live2D

## 关于
vuepress看板娘插件，修改于https://github.com/QiShaoXuan/vuepress-plugin-cat.git，增加了点击回到顶部功能


## 使用

1. `npm install git+https://github.com/stack-stark/vuepress-plugin-live2D.git`

2. add in `config.js`

```js
module.exports = {
  plugins: [
    "vuepress-plugin-live2D",
  ]
}
```

## Tip

- 插件支持 `vuepress 1.x`
